Linux Cheat Sheet for Data Engineers

1. System Information

pwd                 # Present working directory
uname -a            # System/kernel information
whoami              # Current user
history             # Command history

2. Directory Operations

mkdir folder_name               # Create directory
mkdir -p a/b/c                  # Create nested directories

cd folder_name                  # Change directory
cd ..                           # Move one level up
cd ~                            # Home directory
cd                              # Home directory

3. Listing Files

ls                              # List files

ls -l                           # Long format
ls -a                           # Hidden files
ls -al                          # Hidden + long format
ls -t                           # Sort by modified time
ls -r                           # Reverse order
ls -S                           # Sort by file size

4. File Creation & Editing

touch file.txt                  # Create empty file

nano file.txt                   # Open Nano editor
vi file.txt                     # Open Vim editor

5. File Viewing

cat file.txt                    # Display file content

head file.txt                   # First 10 lines
head -20 file.txt               # First 20 lines

tail file.txt                   # Last 10 lines
tail -f file.txt                # Live monitoring

6. Copy, Move & Rename

cp file1.txt backup.txt         # Copy file

cp -r dir1 dir2                 # Copy directory

mv old.txt new.txt              # Rename file

mv file.txt /target/path        # Move file

7. Delete Operations

rm file.txt                     # Delete file

rm -r folder                    # Delete directory

rm -rf folder                   # Force delete directory

rm *.txt                        # Delete all txt files

8. Append Data

cat file1.txt >> file2.txt

echo "Hello World" >> file.txt

echo "Data Engineering" > file.txt

9. Install Packages

sudo apt-get update

sudo apt-get install package_name

Example:

sudo apt-get install wget

10. Links

Hard Link

ln original.txt hardlink.txt

Soft Link (Shortcut)

ln -s original.txt shortcut.txt

11. Bash Configuration

nano ~/.bashrc

Reload:

source ~/.bashrc

12. Aliases

alias ll='ls -al'

alias gs='git status'

View aliases:

alias

13. Background Execution

nohup python app.py &

Output goes to:

nohup.out

14. Process Monitoring

top                         # Live processes

ps                          # Current shell processes

ps -aux                     # All running processes

Kill process:

kill -9 <PID>

Example:

kill -9 12345

15. Download Files

wget "https://example.com/file.zip"

16. Disk Usage

df -h                        # Available disk space

du -sh filename              # File/folder size

Examples:

du -sh data.csv

du -sh logs/

17. Memory Usage

free -m                      # Memory in MB

free -g                      # Memory in GB

18. Compression

ZIP

zip file.zip file.txt

zip -r project.zip project/

UNZIP

unzip project.zip

19. TAR Compression

Create

tar -czvf backup.tar.gz folder/

Options:

  • c = create
  • z = gzip
  • v = verbose
  • f = file

Extract

tar -zxvf backup.tar.gz

20. Word Count

wc file.txt

Specific counts:

wc -w file.txt      # Words

wc -l file.txt      # Lines

wc -c file.txt      # Characters

21. Sorting

sort data.txt

Reverse:

sort -r data.txt

Numeric:

sort -n data.txt

Numeric Reverse:

sort -nr data.txt

22. Grep (Most Important for Data Engineers)

Basic:

grep ERROR logfile.log

Ignore case:

grep -i error logfile.log

Search all txt files:

grep "data" *.txt

Recursive search:

grep -r "spark" .

Regex search:

grep "^[0-9]" data.txt

Common Examples:

grep "FAILED" application.log

grep -i "exception" application.log

grep -r "password" .

23. Find Command

Find file:

find . -name file.txt

Find all txt files:

find . -name "*.txt"

Find empty files:

find . -empty

Delete all txt files:

find . -name "*.txt" -delete

Find directories:

find . -type d

Find files:

find . -type f

24. AWK (Very Important for ETL/Data Engineering)

Sample File:

101 John 50000
102 Alice 60000
103 Bob 70000

Print entire row:

awk '{print}' data.txt

Print first column:

awk '{print $1}' data.txt

Print second column:

awk '{print $2}' data.txt

Print multiple columns:

awk '{print $1,$3}' data.txt

Filter records:

awk '$3 > 60000' data.txt

Sum column:

awk '{sum+=$3} END {print sum}' data.txt

Average:

awk '{sum+=$3} END {print sum/NR}' data.txt

25. File Permissions

r = 4
w = 2
x = 1

Calculation:

7 = rwx = 4+2+1

6 = rw- = 4+2

5 = r-x = 4+1

4 = r--

Examples:

chmod 777 file.txt

chmod 755 script.sh

chmod 644 data.csv

chmod 600 secret.txt

Permission Format:

Owner | Group | Others

755

Owner  = rwx (7)
Group  = r-x (5)
Others = r-x (5)

26. Cron Jobs (Scheduling)

Cron Format:

* * * * *
| | | | |
| | | | +---- Day of Week (0-7)
| | | +------ Month (1-12)
| | +-------- Day of Month (1-31)
| +---------- Hour (0-23)
+------------ Minute (0-59)

Open Cron Editor

crontab -e

View Cron Jobs

crontab -l

Delete Cron Jobs

crontab -r

Examples

Run every minute:

* * * * * /home/user/script.sh

Every day at 10 PM:

0 22 * * * /home/user/script.sh

Every Monday at 9 AM:

0 9 * * 1 /home/user/script.sh

Every 5 minutes:

*/5 * * * * /home/user/script.sh
